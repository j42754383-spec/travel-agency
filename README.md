# FIS Travel Agency - Sitio Web

Proyecto académico ficticio para la Fundación Internacional de Servicios (FIS).

## 📋 Descripción

Sitio web profesional de agencia de viajes con contenido basado en la historia y valores de FIS.

**⚠️ AVISO IMPORTANTE**: Este es un proyecto académico ficticio para propósitos educativos.

## 🎯 Características

- ✅ Diseño responsive (mobile, tablet, desktop)
- ✅ Navegación interactiva con smooth scroll
- ✅ Tabs para secciones "Sobre Nosotros" y "Ofertas"
- ✅ 6 destinos populares con precios
- ✅ Servicios incluidos (Vuelos, Alojamiento, etc)
- ✅ Testimonios de clientes
- ✅ Formulario de contacto funcional
- ✅ Footer con información legal

## 🚀 Inicio Rápido

### Clonar el repositorio
```bash
git clone https://github.com/j42754383-spec/travel-agency.git
cd travel-agency
```

### Ejecutar localmente
Simplemente abre `index.html` en tu navegador.

### Publicar en GitHub Pages
1. Ve a **Settings** del repositorio
2. Baja a **Pages**
3. Selecciona `main` como rama
4. Tu sitio estará disponible en: `https://j42754383-spec.github.io/travel-agency/`

## 📁 Estructura de Carpetas

```
travel-agency/
├── index.html      (HTML principal)
├── styles.css      (Estilos CSS)
├── script.js       (Funcionalidad JavaScript)
├── README.md       (Este archivo)
└── assets/
    └── logo.png    (Logo de FIS - agregar aquí)
```

## 🖼️ Agregar Logo

1. Crea una carpeta `assets` en la raíz
2. Guarda tu logo como `logo.png` en esa carpeta
3. El código ya busca automáticamente en `./assets/logo.png`

**Recomendación**: Usa la imagen circular con "FUNDACIÓN INTERNACIONAL DE SERVICIOS"

## 📝 Secciones

- **Navbar**: Menú de navegación fijo
- **Hero**: Banner principal con CTA
- **Sobre Nosotros**: Historia, valores y diferenciación (con tabs)
- **Destinos**: 6 destinos populares
- **Ofertas**: Verano, Invierno, Última Hora (con tabs)
- **Servicios**: 6 servicios principales
- **Testimonios**: Opiniones de clientes
- **Contacto**: Formulario y datos
- **Footer**: Información legal
